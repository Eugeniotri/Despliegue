Comando Qué hace Ejemplo
git init Inicia un repositorio Git git init
git status Muestra el estado del repositorio git status
git add . Añade todos los cambios al staging git add .
git add archivo Añade un archivo concreto git add README.md
git commit -m "mensaje" Guarda una versión en el historial git commit -m "primer commit"
git log --oneline Muestra historial resumido git log --oneline
git branch Lista ramas git branch
git checkout -b rama Crea y cambia a una rama git checkout -b nueva-seccion
git checkout rama Cambia de rama git checkout main
git merge rama Fusiona una rama con la actual git merge nueva-seccion
git branch -d rama Elimina una rama fusionada git branch -d nueva-seccion
git remote add origin URL Conecta el repo local con GitHub git remote add origin https://github.com/usuario/repo.git
git push -u origin main Sube la rama principal y la vincula git push -u origin main
